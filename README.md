## 📌 Sobre o projeto

Sistema que analisa dados de alunos, identificando o melhor e o pior desempenho com base nas notas inseridas.

---

## ⚙️ Funcionalidades

✔ Cadastro de alunos  
✔ Registro de notas  
✔ Identificação do melhor aluno  
✔ Identificação do pior aluno  
✔ Comparação de desempenho  

---

## 🧠 Conceitos aplicados

- Variáveis compostas  
- Estruturas de repetição  
- Condicionais  
- Comparação de dados  
- Lógica de análise  

---

## 🎯 Objetivo

Simular análise de dados simples, preparando base para estudos futuros em análise de dados.

---

## 🚀 Aprendizados

Desenvolvimento de lógica voltada à análise e comparação de informações.
